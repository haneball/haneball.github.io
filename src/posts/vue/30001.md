## 概述

BackTop 是一个返回页面顶部的快捷按钮。

## 简单实现

### 样式

BackTop 按钮只需要一个 `div`，内部有一个 svg 三角箭头和一个通过 `props` 传入的自定义按钮文本，默认为“顶部”。

显然，当页面向下滚动时，我们希望 BackTop 按钮能够固定在某一个位置，因此使用 `fixed` 定位，且距离页面底、右均为 `120px`。

<!-- more -->

```vue
<template>
  <div class="back-top-btn">
    <svg width="12" height="12">
      <polygon points="0,12 12,12 6,6" fill="#FFF" stroke="none"></polygon>
    </svg>
    <span>{{ text }}</span>
  </div>
</template>

<script>
export default {
  props: {
    text: {
      type: String,
      default: '顶部'
    }
  }
}
</script>

<style lang="scss" scoped>
.back-top-btn {
  width: 48px;
  position: fixed;
  bottom: 120px;
  left: 120px;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  padding: 8px 0;
  font-size: 12px;
  border: 2px solid #000;
  user-select: none;
}
</style>
```

### 逻辑

实现回到顶部非常简单，只需要将 `scrollTop` 设置为 0 即可。但是，这样会表现为页面“闪”到了顶部，效果不理想。

我们可以使用 `requestAnimationFrame` 来写一个简单的 JS 滚动动画。当距离顶部的长度 `distance` 大于 0 才会触发动画，接着递归调用 `requestAnimationFrame` ，以 `100px` 为间距不断向上滚动，直到 `distance` 小于等于 0。最后在组件上设置 `@click=backTop` 以调用滚动方法。

```vue
<script>
export default {
  methods: {
    backTop () {
      let timer = null
      cancelAnimationFrame(timer)
      timer = requestAnimationFrame(function scrollAnime () {
        let distance = document.body.scrollTop || document.documentElement.scrollTop
        if (distance > 0) {
          scrollTo(0, distance - 100)
          timer = requestAnimationFrame(scrollAnime)
        } else {
          cancelAnimationFrame(timer)
        }
      })
    }
  }
}
</script>
```

除了回到顶部，我们还需要实现，滚动距离较短时隐藏 BackTop 按钮。

页面挂载后，添加一个事件监听器，回调函数 `handleScroll` 控制显示与否。然后在组件上设置 `v-show="showBtn"`。

```vue
<script>
export default {
  data () {
    return {
      showBtn: false
    }
  },
  mounted () {
    window.addEventListener('scroll', this.handleScroll)
  },
  beforeDestroy () {
    window.removeEventListener('scroll', this.handleScroll)
  }，
  methods: {
    handleScroll () {
      let scrollTop = document.documentElement.scrollTop
      if (scrollTop >= 400) {
        this.showBtn = true
      } else {
        this.showBtn = false
      }
    }
  }
}
</script>
```

## 进阶优化

### 定位

以上基本实现了 BackTop 按钮的功能，但存在一个可以改进的地方。

为了方便大部分人的操作，BackTop 按钮通常会固定在页面的右下角。以上例子直接在组件上使用了 `fixed` 定位，考虑到我们可能会给 BackTop 按钮设置更为复杂的样式，不妨尝试将其定位属性抽离，在外层的 `div` 完成定位。

示意图如下

![BackTop 定位](/assets/images/case/BackTop.jpg)

首先，改变组件的整体的结构

```vue
<template>
  <div v-show="showBtn" class="back-top-outer">
    <div class="back-top-inner">
      <div class="back-top-wrap">
        <div class="back-top-btn" @click="backTop">
          <svg width="12" height="12">
            <polygon points="0,12 12,12 6,6" fill="#FFF" stroke="none"></polygon>
          </svg>
          <span>{{ text }}</span>
        </div>
      </div>
    </div>
  </div>
</template>
```

其次，设置相应的样式，以实现定位
- back-top-outer：`fixed` 定位，覆盖整个页面，设置 `pointer-events: none;` 不触发事件，通过左右的 `padding` 实现水平定位
- back-top-inner：`relative` 定位，用于填充 back-top-outer
- back-top-wrap：`absolute` 定位，`left: 100%` 紧贴 back-top-inner，通过 `bottom` 实现垂直定位
- back-top-btn：设置 `pointer-events: auto;` 可以触发事件

```vue
<style lang="scss" scoped>
.back-top-outer {
  position: fixed;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  padding: 0 96px;
  background-color: transparent;
  z-index: 1999;
  pointer-events: none;
}

.back-top-inner {
  position: relative;
  width: 100%;
  height: 100%;
}

.back-top-wrap {
  position: absolute;
  bottom: 120px;
  left: 100%;
}

.back-top-btn {
  width: 48px;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  padding: 8px 0;
  font-size: 12px;
  border: 2px solid #000;
  user-select: none;
}
</style>
```

### 样式

我们已经实现了 BackTop 按钮的逻辑以及定位，为了组件变得更美观，我们可以为 BackTop 按钮设置字体、背景、边框的样式，或是添加一个 [fade 淡入淡出](../css/20001.md#fade-淡入淡出) 动画。

最终代码如下

```vue
<template>
  <div id="back-top-btn">
    <div v-show="showBtn" class="back-top-outer">
      <div class="back-top-inner">
        <div class="back-top-wrap">
          <div class="back-top-btn" @click="backTop">
            <svg width="12" height="12">
              <polygon points="0,12 12,12 6,6" fill="#FFF" stroke="none"></polygon>
            </svg>
            <span>{{ text }}</span>
          </div>
        </div>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  props: {
    text: {
      type: String,
      default: '顶部'
    }
  },
  data () {
    return {
      showBtn: false
    }
  },
  mounted () {
    window.addEventListener('scroll', this.handleScroll)
  },
  methods: {
    handleScroll () {
      let scrollTop = document.documentElement.scrollTop
      if (scrollTop >= 400) {
        this.showBtn = true
      } else {
        this.showBtn = false
      }
    },
    backTop () {
      let timer = null
      cancelAnimationFrame(timer)
      timer = requestAnimationFrame(function scrollAnime () {
        let distance = document.body.scrollTop || document.documentElement.scrollTop
        if (distance > 0) {
          scrollTo(0, distance - 100)
          timer = requestAnimationFrame(scrollAnime)
        } else {
          cancelAnimationFrame(timer)
        }
      })
    }
  },
  beforeDestroy () {
    window.removeEventListener('scroll', this.handleScroll)
  }
}
</script>

  left: 0;

  }
}

  from {
    opacity: 0;
  }
  to {
    opacity: 100%;
  }
}
</style>
```
